# Meme Generator 🎭

A simple and fun meme generator built with Next.js! 🚀

## Features ✨
- Responsive and fast ⚡

## Getting Started 🛠

### 1. Clone the Repo 📂
```bash
git clone https://github.com/yourusername/meme-generator.git
cd meme-generator
```

### 2. Install Dependencies 📦
```bash
npm install
```

### 3. Run the App 🚀
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack 💻
- Next.js
- Tailwind CSS
